# CLI youtube downloader

This command line interface allows you to download YouTube playlists or videos and convert them to mp3 files.

## Getting Started

### Requirements
- [Node.js](https://nodejs.org)

### Install
```
$ npm install --global @dedeard/dyp
```

## Usage

```
$ dyp
```
